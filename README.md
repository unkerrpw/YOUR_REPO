# BallBlast 🎮

2D mobile ball game built with Godot 4.2

## Build

Push to `main` → GitHub Actions auto-builds APK → download from **Actions** tab.

### Controls
- **Tap** — speed boost
- **Swipe** — change direction  
- **Tap (full energy)** — Super Blast 💥

## Download APK
Go to **Actions → latest run → Artifacts → BallBlast-APK**
